# Robot System Learning 2026

机器人系统学习笔记与项目记录，涵盖飞控、电机控制、ROS2、Android 等方向。

## 目录结构

```
├── flight-control/        # 飞行控制相关学习
├── foc-motor-control/     # FOC 电机控制相关学习
├── linux-ros2-bridge/     # Linux 与 ROS2 桥接相关学习
├── android-system-notes/  # Android 系统相关学习
└── docs/                  # 通用文档资料
```

## 各模块说明

### flight-control
飞控系统学习，包括姿态解算、PID 控制、传感器融合等。

### foc-motor-control
FOC（磁场定向控制）电机驱动学习，包括 Clark/Park 变换、SVPWM、无感控制等。

### linux-ros2-bridge
Linux 系统与 ROS2 开发学习，包括节点通信、话题/服务/动作、Launch 文件等。

### android-system-notes
Android 系统开发学习，包括 Framework、HAL、Binder 通信等。

### docs
通用文档、参考资料、学习路线图等。

## 环境要求

根据各模块需求不同，可能涉及：
- PX4 / ArduPilot 飞控固件
- STM32 / ESP32 开发环境
- ROS2 Humble / Iron
- Android AOSP 源码环境
